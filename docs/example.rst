Example
-------

Demonstration: Using
`pandoc-numbering-sample.txt <https://raw.githubusercontent.com/chdemko/pandoc-numbering/develop/docs/images/pandoc-numbering-sample.txt>`__
as input gives output files in
`pdf <https://raw.githubusercontent.com/chdemko/pandoc-numbering/develop/docs/images/pandoc-numbering-sample.pdf>`__,
`tex <https://raw.githubusercontent.com/chdemko/pandoc-numbering/develop/docs/images/pandoc-numbering-sample.tex>`__,
`html <https://raw.githubusercontent.com/chdemko/pandoc-numbering/develop/docs/images/pandoc-numbering-sample.html>`__,
`epub <https://raw.githubusercontent.com/chdemko/pandoc-numbering/develop/docs/images/pandoc-numbering-sample.epub>`__
and other formats.
